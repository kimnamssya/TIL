## Data 분석을 위한 Python

##### 1. Python - String

* 문자열은 리스트처럼 인덱싱이 가능

```python
s1 = "abcd"

s1[0] : a 
s1[1] : b
s1[2] : c
s1[3] : d

s1[-1] : d
s1[-2] : c
s1[-3] : b
s1[-4] : a
```



* 문자열은 리스트처럼 슬라이싱이 가능

```python
s1 = "abcd"

s1[1:2] = b
s1[0:3] = abc
s1[:3] = abc
s1[1:] = bcd
s1[:] = abcd
```
