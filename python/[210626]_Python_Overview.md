#### 2. 기본 문법

##### 2.1 들여쓰기 (Indent)

```python
>>> if age > 19
	    print("성인입니다.")
```

* 일반적으로 >>> (Python prompt)의 첫 칸부터 명령어 입력
* `if`, `for`, `class`, `def`등 여러 문장이 한 블록을 구성하는 경우 들여쓰기(Tab 또는 4칸 공백)를 통해 한 블록임을 표시

```python
>>> 1+2
>>> 1  +   2
```

* 개행과 들여쓰기를 규칙에 맡도록 해야하지만 공백은 무관함
* 들여쓰기 에러 시 `SyntaxError: unexpected indent` 에러 발생
